# DCGAN Streamlit Demo

Interactive web application for demonstrating DCGAN image generation.

## Setup

1. **Install Streamlit:**
   ```bash
   pip install streamlit
   ```

2. **Run the app:**
   ```bash
   streamlit run app.py
   ```

3. **Open in browser:**
   - The app will automatically open at `http://localhost:8501`
